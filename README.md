# Dashboard_Gatitopetshop_powerbi
Nesse Desafio foi desenvolvido um dashboard para o pet shop gatito, esse petshop eles so trabalhavam com bases de dados do excel e queriam uma opção mais visual e organizada.
Respondemos perguntas como:

📊 Qual é o faturamento total? 
📊 Qual é a média de pets? 
📊 Qual é a quantidade de vendas? 
📊 Qual bairro vende mais?
📊 Quem compra mais ?
📊Qual data vendemos mais?

Durante o desenvolvimento, passamos por algumas etapas importantes:

🔹 Importação dos dados a partir do Excel 
 
🔹 Tratamento e limpeza dos dados no Power Query 

🔹 Criação do layout no figma e importação para o Power BI como template.

🔹 Adicionamos um filtro para ficar de fácil acesso a busca.
